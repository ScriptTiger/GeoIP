# GeoIP
A Java GUI client for the [IP_Search](https://github.com/ScriptTiger/goIP) REST API.

**GeoIP-8.jar**  
Supported by JDK/JRE 8+ (suitable for most users).

**GeoIP-17.jar**  
Supported by JDK/JRE 17+.
